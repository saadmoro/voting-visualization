Data Visualization of Youth Voter Turnout in Presidential Elections 1964-2016
@author Saad Moro

This repository contains R/ggplot2 code that creates a visualization of voting turnout in a number of U.S. presidential elections, stratified by age. The data used is sourced from the United States Census Bureau, Historical Reported Voting Rates Table A-1, and is contained here. A png file is saved upon completion of the R code.
